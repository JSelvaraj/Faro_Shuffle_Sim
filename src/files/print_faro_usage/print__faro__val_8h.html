Function to generate stacscheck-compliant output.

This function called once per data-item. A complete deck requires (2+size) calls to output a deck's data. For example, a deck of 52 cards consists of:

  • one call for output type (PREFIX_START, PREFIX_IN, PREFIX_OUT)
  • 52 calls for the card contents (CARD_VAL)
  • one call to terminate the output with "EoD" (SUFFIX)

EXAMPLE USAGE:

  • print_faro_val("", PREFIX_IN); // outputs " IN: "
  • print_faro_val(card_val, CARD_VAL); // eg. if card_val <– "5H\0", outputs "5H "
  • print_faro_val("", SUFFIX); // OUTPUTS "EoD\n"
Parameters
